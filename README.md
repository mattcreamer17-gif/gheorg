# Gheorg landing page

A single-page note prepared for the Gheorg team (Dr Louise Metcalf, Dana Klein) at eMerge Americas 2026, booth SS24. The page uses Gheorg's own design vocabulary taken from the live site: Rubik typography, the navy-teal-coral palette pulled from their compiled Squarespace CSS, rounded pill buttons, and soft shadows. It carries the pitch body from fit-analysis section 7 (stack-layer read of where AI belongs in the product, Feeld as the shape-matched case study, the Ethics Engine paper for clinical AI governance, NLP Logix partnership note with the mandatory credibility stats), and a single CTA to book thirty minutes with Matt Creamer, Eduba's CRO. Eduba branding sits only in the footer and the named case study context.
